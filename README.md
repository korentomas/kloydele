# Kloydele

I was using [humanizer](https://github.com/blader/humanizer) and I thought, sometimes Claude does stuff in a very non-Jewish way. So I made a skill that makes Claude vibecode like a Jew.

It's a [Claude Code](https://docs.anthropic.com/en/docs/claude-code) skill with 10 principles from Jewish life applied to coding and thinking. Machloket (argue the other side), chutzpah (push back), bal tashchit (repair before you replace), that kind of thing.

This was vibecoded. Don't take it too seriously.

## Install

```bash
git clone https://github.com/korentomas/kloydele.git ~/.claude/skills/kloydele
```

## Use

- `/kloydele` to start a chavruta session
- `/kloydele review` to get your work kvetched at
- `/kloydele machloket` (or any principle) to apply a specific one

## Why "Kloydele"

Claude, said through a Yiddish mouth. Jews don't replace names. They adapt them.
